# Lightclip
A clipping application made in C# WPF
# Instructions
- Ctrl + Alt + G to clip the last 30 seconds (Changeable in settings)
- Clips saved in your Videos folder (Changeable in settings)
# Libraries
- [FFMpegCore](https://github.com/rosenbjerg/FFMpegCore)
- [Hardcodet.NotifyIcon.Wpf](https://github.com/hardcodet/wpf-notifyicon)
- [ScreenRecorderLib](https://github.com/sskodje/ScreenRecorderLib)
